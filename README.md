# hubot-sonos
Control your Sonos device with Hubot

## Install 

`npm install hubot-sonos`

## Usage


```
 Commands:
   hubot what's playing - Return current track
   hubot play  - play
   hubot pause - pause
   hubot next  - next
   hubot previous - back
   hubot back     - back
   hubot volume <0-100> - set volume

 Configuration:
   HUBOT_SONOS_HOST - the address of your Sonos device
 ```
